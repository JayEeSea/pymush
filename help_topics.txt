Available commands:

/quit - Disconnect from the server
/who - List all connected users
/addchannel <channel_name> <display_name> - Add a new chat channel
/listchannels - List all available chat channels
/removechannel <channel_name> - Remove an existing chat channel
/look - Look around the current room
/go <direction> - Move to a new room
/help - Show this help message
/help <command> - Show detailed help for the specified command
